# rustdesk-id-changer-linux

Rustdesk ID Changer for Linux

This is a Linux port from RustDesk ID Changer : https://github.com/abdullah-erturk/RustDesk-ID-Changer

## Getting starts

* Download and run script with sudo

```bash
sudo sh -c "$(curl -fsSL https://raw.githubusercontent.com/aruyu/rustdesk-id-changer-linux/master/rustdesk_id_changer.sh)"
```

* Clone and run script with sudo

```bash
git clone https://github.com/aruyu/rustdesk-id-changer-linux.git
sudo ./rustdesk-id-changer-linux/rustdesk_id_changer.sh
```

### Select options

1. Set RustDesk ID to hostname.

2. Set RustDesk ID to 9-digit random numbers.

3. Set RustDesk ID to the custom value.
